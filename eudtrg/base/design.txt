Base : 트리거 관련
 - addressable   : STR 단락에 넣을 수 있는 덩어리들(Addressable), 이들을 가르키는 포인터(Addr) 정의
 - expr          : Addr가 포함된 수식을 처리하는 Expr 클래스 정의. 또한 Expr 클래스의 값을 계산하는 Evaluate 함수 제공

 - trigger       : 트리거 구조체 (Trigger, Conditions, Actions) 정의
 - bytedump      : 바이너리 데이터를 그대로 덤핑할 수 있는 ByteDump 클래스.


Payload : STR 단락에 들어갈 데이터 제작
 - rlocint       : Relocatable Int
 - depgraph      : Dependency Graph를 구성한다. Dead Code Elimination이 자동적으로 되도록 함
 - payload       : Addressable에서 STR 단락에 바로 넣을 수 있는 형태로 (data, PRT, ORT)로 변환


mpqrw : 맵 관련 api
 - mpqapi        : SFmpq를 이용해서 scx 파일을 연다
 - chktok        : chk 파일을 다룬다


Inject : chk 파일에 Payload 인젝션 & 잡다한 처리.
 - injgen        : Injector 제작
 - eudenable     : EUD Enabler 제작 / 삽입 (아직 연구되어야함)
