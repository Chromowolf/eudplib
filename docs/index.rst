Welcome to eudtrg documentation!
================================

Documentation for eudtrg. Don't confuse it with EUDTrig.

eudtrg is a tool for generating Self-modifying trigger systematically. eudtrg
works on top of EUD Actions, so EUD Action Enabler will be needed when running
eudtrg-generated maps.
